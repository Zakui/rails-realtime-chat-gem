# ChatEngine
Chat Engine for real time chat between users

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'chat_engine'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install chat_engine
```
then run
```bash
$ rails g chat_engine:install
```
